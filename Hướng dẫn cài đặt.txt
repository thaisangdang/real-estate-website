--------------------------------------Lưu ý với Visual Studio 2015---------------------
Khi build project sẽ bị lỗi file tsc.exe, là file biên dịch mã nguồn TypeScript
=> Cách khắc phục: đóng Visual Studio 2015, tải và cài đặt phần mở rộng "TypeScript 2.2 for visual Studio 2015" từ menu Tools -> Extensions and Updates
Hoặc tải từ link https://typescriptteam.gallerycdn.vsassets.io/extensions/typescriptteam/typescript22forvisualstudio2015/2.2/1487789113613/249189/1/TypeScript_Dev14Full.exe

Sau khi cài xong Extension thì mở lại project bằng visual studio 2015 sẽ biên dịch và chạy project bình thường.
